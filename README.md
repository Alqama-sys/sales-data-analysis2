# Sales Data Analysis

## Project Objective

To analyze sales data and identify revenue trends, top-performing products, and monthly sales performance using Python.

## Dataset

The dataset contains order-level sales information including product, category, quantity, price, and order date.

## Tools & Technologies

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights

- Laptops are the highest revenue-generating product.
- Electronics contribute the most to total sales.
- Monthly sales show an upward trend, indicating business growth.

## Conclusion

This project demonstrates basic data analysis skills including data cleaning, feature engineering, aggregation, and visualization.
